# 🌱 AgriNova - Smart Crop Management System

AgriNova is an AI-powered crop management platform that helps farmers and agricultural coordinators track field progress, manage resources, and predict risks using machine learning.

## 🚀 Features

### Core Functionality
- **Multi-role Authentication** (Admin & Field Agent)
- **Field Management** with full CRUD operations
- **Real-time Field Updates** with stage tracking
- **Automated Status Calculation** (Active, At Risk, Completed)
- **Interactive Dashboards** with charts and analytics

### AI/ML Integration
- **Risk Prediction Models** using TensorFlow.js
- **Intelligent Recommendations** based on field data
- **Stage Transition Suggestions**
- **Confidence Scoring** for predictions

### Smart Features
- **Weather Data Integration**
- **Field Health Scoring**
- **Automated Risk Assessment**
- **Predictive Harvest Dates**

## 🛠️ Technology Stack

### Frontend
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Chart.js for visualizations
- React Hot Toast for notifications

### Backend
- Next.js API Routes
- Prisma ORM
- PostgreSQL
- JWT Authentication
- bcrypt for password hashing

### AI/ML
- TensorFlow.js
- Custom risk prediction models
- Rule-based recommendation engine

## 📋 Prerequisites

- Node.js 18+ 
- PostgreSQL 14+
- npm or yarn package manager
